Real-Time Distributed Flocking Control

I propose the aerial swarm flocking as a probabilistic inference and solve it in a distributed manner using mean-field variational inference. The inference procedure takes place in a distributed manner in each drone using its local observations to calculate the control actions. This allows achieving dynamically feasible real-time flocking control with convergence guarantees borrowed from graphical games.
